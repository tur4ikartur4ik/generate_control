# Приложение для генерации индивидуальных саданий по теме "Машинное представление dещественных чисел".

## Описание
Приложение на языке C генерирует индивидуальные задания по теме машинного представления вещественных чисел (IEEE-754).

Для каждого варианта создаются два файла .md:

1. В каталоге "Задания": файл с исходными числами (для студента),
2. В каталоге "Проверка": файл с исходными числами, машинными представлениями и ошибками округления (для преподавателя).

### Входные данные
Данные считываются с файла с содержимым:

```
n - количество вариантов
k - количество заданий в вариантах
bit_qt - разрядность чисел в тестовых заданиях (32/64/128)
a, b - диапозон вещественных чисел в заданиях
p - количество знаков после запятой в числах
```
**Пример входного файла:**
```
10
5
32
-99.9 99.9
3
```

### Выходные данные

Создаются два каталога:
* Задания
* Проверка

В каждом каталоге создаются n файлов c k вариантами в формате markdown.

**Пример выходных файлов:**
* Для студента:
# Вариант 1.

## Приведите представленные числа к машинному представлению 32 бит.

| N | x |
|---:|---:|
| 1 | 67.970 |
| 2 | -21.102 |
| 3 | 56.563 |
| 4 | 59.628 |
| 5 | 82.247 |

* Для преподавателя:
# Вариант 1
| N | X | машинное представление 32 бит | Ошибка |
|---:|---:|---:|---:|
| 1 | 67.970 | 01000010100001111111000010100100 | 0.000001220703125 |
| 2 | -21.102| 11000001101010001101000011100101 | 0.000000717163086 |
| 3 | 56.563 | 01000010011000100100000010000011 | 0.000000274658203 |
| 4 | 59.628 | 01000010011011101000001100010010 | 0.000001647949219 |
| 5 | 82.247 | 01000010101001000111111001110111 | 0.000001647949219 |

### Запуск программы (linux mint)
```
gcc generate_control.c -o generate_control -lquadmath
./generate_control
```
